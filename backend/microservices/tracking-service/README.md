# Tracking Service

Микросервис отслеживания отправлений по QR кодам и событиям сканирования.

## Функционал

- Генерация QR кодов для отправлений
- Сканирование отправлений на станциях
- Отслеживание истории сканирований
- Получение информации об отправлении по коду отслеживания
- Просмотр истории событий доставки

## API Endpoints

- `POST /tracking/qrcode` - Генерация QR кода
- `GET /tracking/:code` - Получение информации об отправлении
- `POST /tracking/scan` - Сканирование отправления
- `GET /tracking/:id/events` - Список событий сканирования
- `GET /tracking/:id/history` - История отслеживания

## Интеграция

Этот микросервис взаимодействует с:
- Shipment Service - для получения информации об отправлениях
- Auth Service - для аутентификации пользователей
