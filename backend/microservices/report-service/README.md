# Report Service

Микросервис формирования отчетов и аналитики.

## Функционал

- Получение панели управления (Dashboard)
- Финансовые отчеты
- Сводка по статусам отправлений
- Аналитика по кварталам

## API Endpoints

- `GET /reports/dashboard` - Панель управления
- `GET /reports/finance` - Финансовый отчет
- `GET /reports/status-summary` - Сводка по статусам

## Metrics

### Dashboard
- Всего отправлений
- Активных отправлений
- Завершенных отправлений
- Общая выручка
- Ожидающие платежи

### Finance
- Общая выручка
- Всего платежей
- Ожидающие платежи
- Возвращено средств
- Средняя стоимость заказа

### Status Summary
- Статус
- Количество
- Процент от общего числа
